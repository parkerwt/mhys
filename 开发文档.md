# 周易·数字通行本 (Digital I Ching Divination App) ☯️

## 📝 项目介绍

**“周易·数字通行本”** 是一款基于原生 Web 技术（HTML/CSS/JS/Canvas）开发的纯前端沉浸式占卜应用。
项目严格遵循北宋易学家邵雍创立的**《梅花易数》**（Plum Blossom Numerology）起卦原理。通过将现代前端的哈希算法与古人“心机起卦”的理念结合，实现了兼具现代科技感与国风神秘学的玄学互动体验。

项目不仅提供了华丽的 Canvas 八卦转盘动态推演，还在数据侧实现了《周易》64卦、384爻辞的完整映射，支持一键将卦象结果转化为 Prompt 注入至 AI 大语言模型进行深度占卜解析。

## ✨ 核心特性

1. **数字梅花易数算法**：摒弃传统铜钱起卦的复杂性，采用输入文本（物数）结合时间戳（时数）生成上下卦与动爻，符合“无动不占、随心起卦”的易理内核。
2. **完整三卦推演**：自动计算并展示：
   - **本卦（现状）**：当前事态的基本盘。
   - **互卦（过程）**：隐藏的内因与发展过程（取2,3,4爻和3,4,5爻）。
   - **之卦（结果）**：动爻翻转后的终局之象。
3. **沉浸式 Canvas 互动**：动态八卦双轨转盘（内外盘分别控制），平滑的阻尼旋转动画。
4. **一键 AI 解析协议**：内置高维 Prompt 模板，可一键复制结构化的卦象数据至 ChatGPT / Claude / Gemini 中获取大师级解卦报告。
5. **高度模块化的数据层**：采用 `DataProvider` 类统一调度 JSON 数据结构，内存常驻建立二进制树形索引，查询时间复杂度为 O(1)。

## 🛠 技术栈

- **核心结构**：HTML5, CSS3 变量与 Flexbox 布局
- **业务逻辑**：ES6 (Vanilla JavaScript) 无框架依赖，极致轻量
- **图形渲染**：HTML5 Canvas API
- **数据存储**：分离式 JSON 本地数据库 (包含 8 卦、64 卦、384 爻辞及五行/颜色信息)

## 📐 易理算法说明

在 `script.js` 中，起卦引擎遵循以下逻辑：

- **下卦（内卦）取数**：利用用户输入所求之事的 `charCodeAt` 累计值作为“先天数”，对 8 取模，对应先天八卦（乾1兑2离3震4巽5坎6艮7坤8）。
- **上卦（外卦）取数**：在用户点击第二次起卦时，抓取系统毫秒级时间戳的尾数对 8 取模。
- **变爻计算**：动爻决定了事物的变化规律，计算公式严格遵循 `(下卦数 + 上卦数 + 当时时辰数) % 6`，余数（1~6）即为动爻位。
- **动爻翻转**：二进制字符串中（例如天风姤 `111011`），找到对应的动爻位置进行 `0变1` 或 `1变0`，推演出之卦（变卦）。

## 🚀 本地运行与部署

项目为纯前端静态应用，但由于浏览器的同源策略（CORS），本地直接双击打开 `index.html` 可能会导致 JSON 数据无法加载（`fetch` API 报错）。

**推荐运行方式：**

1. **使用 VS Code**：安装 `Live Server` 插件，右键 `index.html` 选择 "Open with Live Server"。
2. **使用 Node.js**：
   ```bash
   npx serve .
   ```
3. **使用 Python**：
   ```bash
   python -m http.server 8000
   ```
   然后访问 `http://localhost:8000` 即可体验。
